# artstore20160115

##安裝：
1. git clone https://github.com/afgnsu/artstore20160115.git
2. cd artstore20160115
3. bundle               
4. rake db:create db:migrate db:seed
5. rails s
6. 登入 (帳號：a@a.a，密碼：12345678)
